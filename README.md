Service for accounting employees and salaries

## Tech Stack

**Client:** React, TypeScript, Hooks

## Installation

Install marvel-app with yarn

```bash
  yarn install
```

Compiles and hot-reloads for development

```bash
  yarn run start
```

Compiles and minifies for production

```bash
  yarn run build
```
